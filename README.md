#### Project description
The first Hexlet learning project containing 5 games for the user.
#### Minimum requirements
Need Node.js to be installed.
#### Installation and startup instructions
Installation steps:
1. git clone
2. cd ../frontend-project-44
3. npm install
4. npm link

Startup example:
